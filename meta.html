meta tag defines the metadata about an HTMl element
        metadata is data about data.
        This tag typically used for specify charecterset,page description,keywords, author of the document and view port of the settings.
        metadata didn't displayed on the browser.

        Attributes :
        charset - speciefies the charecter encoding(UTF-8)
        
            Ex :  meta charset="UTF-8" 
        

        content - specifies the value associated with the http-equiv
        
            Ex : meta name="keywords" content="html,css,javascript"
        

        http-equiv - provides an HTTP header for the information of the content attribute
        
            Ex : meta http-eqiv="refresh" content=30
        

        name - the name deifnes descriotion,keywords and the author of an HTML element
        
            Ex : meta name="description" content="free web tutorials"
